# Hans Figo Portfolio
Forked From Nikola Tesla Portfolio

## Getting Started
Clone Repository
```sh
git clone https://github.com/iann-mathaiya/nikola-tesla.git
```

Install Dependencies
```sh
pnpm install
```

Development
```sh
pnpm run dev
```

Build
```sh
pnpm run build
```

Preview
```sh
pnpm run preview
```

## Project Structure

Inside of your Astro project, you'll see the following folders and files:

```text
/
├── public/
│   └── favicon.jpg
│   └── social-image.svg
├── src/
│   ├── actions/

│   ├── assets/

│   ├── layouts/
```
